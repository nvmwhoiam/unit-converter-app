# Unit Converter App

A lightweight, responsive web application for converting values between different units of measurement. Supports weight, temperature, length, and speed conversions with a clean, intuitive interface.

## Features

- **Conversion Categories**: Weight, Temperature, Length, Temperature, Digital/ Copmuting, Volume, Time and Speed
- **Real-time Conversion**: Instant results as you type
- **Swap Functionality**: Easily swap "from" and "to" units
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Animated Navigation**: Smooth open/close animations for mobile menu
- **Accessible**: Built with semantic HTML and ARIA attributes

## Tech Stack

- **HTML5:** Core structure and semantic markup
- **CSS3/ SCSS:** Advanced styling with SCSS preprocessing
- **JavaScript:** Dynamic functionality and user interactions
- **Custom Icon Fonts:** Optimized icon delivery system

## Instalation

To run the Unit Converter App locally, follow these steps:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/nvmwhoiam/unit-converter-app.git
   cd unit-converter-app
   ```

2. **Open the index.html file in your web browser**:

   ```bash
   open index.html
   ```

   or simply drag and drop the `index.html` file into your web browser.

3. **No build process or dependencies required!**

## Usage

**Select Conversion Type**: Click on Weight, Temperature, Length, or Speed

**Enter Value**: Type a number in the input field

**Choose Units**: Select the unit to convert from and to using the dropdown menus

**View Result**: The converted value appears instantly

**Swap Units**: Click the swap icon (↔️) to exchange "from" and "to" units
